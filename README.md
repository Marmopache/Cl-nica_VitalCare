## Descripción del Proyecto

Clínica VitalCare es un sistema web desarrollado en Python con Django, diseñado para gestionar pacientes, doctores y horas médicas de manera eficiente. La aplicación permite registrar, modificar, buscar y eliminar datos clínicos, asegurando que solo usuarios autenticados puedan acceder al panel de control. También incluye un panel de administración para gestionar directamente las tablas de la base de datos.

El proyecto integra Bootstrap para una interfaz responsiva y moderna, compatible con dispositivos móviles y de escritorio, siguiendo buenas prácticas de desarrollo Front-End y Back-End.

## Objetivos

- Aplicar fundamentos de Python y Django en un proyecto web real.
- Implementar un sistema CRUD completo para Pacientes, Doctores y Horas Médicas.
- Garantizar autenticación de usuarios para la gestión de la información.
- Desarrollar una interfaz multiplataforma utilizando Bootstrap.
- Configurar un panel de administración para la gestión de la base de datos en PostgreSQL.

## Tecnologías Utilizadas

- **Back-End:** Python 3.x, Django 6.0
- **Base de Datos:** PostgreSQL
- **Front-End:** HTML5, CSS3, Bootstrap 5, JavaScript
- **Control de Versiones:** Git y GitHub
- **Editor de Código:** Visual Studio Code

## Funcionalidades

### Panel de Control

- Registro de Pacientes: nombre, apellido, edad y fecha de nacimiento.
- Registro de Doctores: nombre, apellido, edad, fecha de nacimiento y especialidad.
- Registro de Horas Médicas: asignación de hora, doctor, paciente, fecha y motivo.
- Listado, modificación y eliminación de pacientes, doctores y horas médicas.
- Autenticación de usuarios para garantizar seguridad en el acceso.

### Panel de Administración

- Gestión completa de las tablas de la base de datos.
- Visualización y edición directa de registros.
